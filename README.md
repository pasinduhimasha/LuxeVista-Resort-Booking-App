# LuxeVista Resort Booking App

LuxeVista Resort Booking App is a native Android application designed to enhance the guest experience at a luxury beachfront resort. The app allows users to browse rooms, book accommodations, reserve resort services, and manage their bookings through a clean and user-friendly mobile interface.

This project demonstrates Android development best practices using Java, Firebase, and modern UI components.

---

## Features

- User registration and login with Firebase Authentication
- Browse available hotel rooms
- Book rooms with date and time selection
- Reserve resort services (spa, dining, cabanas, etc.)
- View booking and reservation history
- Search rooms by name
- Notifications for user actions
- Secure logout and session management

---

## Technologies Used

- **Platform:** Android
- **Language:** Java
- **IDE:** Android Studio
- **Backend:** Firebase Authentication & Firebase Realtime Database
- **UI Components:** RecyclerView, CardView, ConstraintLayout
- **Design Assets:** SVG icons and images

---

## Application Modules

- Authentication (Register / Login / Logout)
- Room Booking Management
- Service Reservation Management
- Search Functionality
- Booking History & Service History
- User Dashboard

## Installation

1. Clone the repository
2. Open the project in **Android Studio**
3. Connect Firebase using your own `google-services.json`
4. Build and run the application on an Android device or emulator
